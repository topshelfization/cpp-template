# Template C++ Project

**Version 0.0.0**

## Details

### _Info_

Simple c++ template project with a basic Makefile and README.

### _Layout_

Possible folder structure of a project using this makefile:

```sh
/TEMPLATE-DIRECTORY
│   .gitignore
│   Makefile
│   README.md
│
├───bin
├───include
├───lib
└───src
        main.cpp
```

### _Compilation_

From the root directory of the project.

To compile:

```sh
mingw32-make
```
To remove all the generated files:

```sh
mingw32-make clean
```
